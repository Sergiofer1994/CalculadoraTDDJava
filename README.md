# 🧮 Java Calculator - TDD Practice

## 📌 Project Description

This project is a simple **Calculator application built in Java** using the **Test Driven Development (TDD)** approach.

The calculator performs the four basic mathematical operations:

- Addition
- Subtraction
- Multiplication
- Division

The project includes **unit tests using JUnit** to verify that each method works correctly.

The application also prints the **full operations and results in the console**.
